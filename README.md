# 🌈 הא-ב הקסום

אפליקציה צבעונית ללימוד ה-א-ב לילדים בגיל הרך — עם הקראה קולית בעברית, ציורים,
משחקים, ומערכת ניקוד וגביעים. עובדת בכל דפדפן, גם ללא התקנה.

**🌐 אתר חי:** https://estishemtov.github.io/learn-alpha/

## מה יש באפליקציה
- **🔤 לומדים אותיות** — כרטיסייה לכל אות עם ציור, מילה, והקראה בעברית.
- **🎨 כל האותיות** — לוח כל הא-ב; לחיצה מקריאה את האות והמילה.
- **👂 מצא את האות** — שומעים אות ובוחרים אותה.
- **🖼️ איזו אות ראשונה?** — רואים ציור ובוחרים באיזו אות הוא מתחיל.
- **✏️ חברו ציור לאות** — מותחים קו בין ציור לאות הראשונה שלו (5 בכל לוח).
- **🏆 האליפות שלי** — צוברים נקודה על כל הצלחה; ב-100 נקודות מקבלים גביע.

## קבצים
- `index.html` — כל האפליקציה (HTML/CSS/JS בקובץ אחד).
- `audio-data.js` — כל קטעי הקול העבריים מוטמעים כ-base64 (עובד גם ללא אינטרנט).
- `recordings/` — הקלצות קול אישיות שמחליפות קטעים (ראו `recordings/README-שמות-קבצים.md`).
- `build-audio.js` — מוריד מחדש את קטעי הקול של כל האותיות והמילים (דורש אינטרנט).
- `build-feedback.js` — מוריד את קטעי המשוב ("כל הכבוד" / "אופס").
- `merge-recordings.js` — מטמיע הקלטות אישיות מתיקיית `recordings/` מעל קטעי ה-TTS.
- `server.js` / `start-server.bat` — שרת מקומי אופציונלי להרצה מ-`http://localhost`.

## 👩‍💻 המשך פיתוח (גם מחשבון / מחשב אחר)

כל מה שצריך נמצא במאגר הזה. כדי להמשיך לערוך:

### 1. לשכפל את הפרויקט
```bash
git clone https://github.com/ESTISHEMTOV/learn-alpha.git
cd learn-alpha
```

### 2. לערוך
כל הלוגיקה, העיצוב והתוכן נמצאים ב-`index.html`. הנתונים (אותיות, מילים, ציורים)
נמצאים במערך `LETTERS` בתוך `<script>`. פותחים את `index.html` בדפדפן כדי לבדוק.

### 3. לפרסם עדכון (GitHub Pages)
האתר החי מתעדכן אוטומטית מכל דחיפה ל-`main`:
```bash
git add -A
git commit -m "התיאור שלך"
git push
```
תוך כ-דקה השינוי עולה ל-https://estishemtov.github.io/learn-alpha/ .
(האתר ציבורי; GitHub Pages מוגדר לענף `main`, שורש הפרויקט.)

### 4. לשנות/להוסיף קטעי קול
- **קול לכל האותיות/מילים:** `node build-audio.js` (מוריד מ-Google TTS בעברית).
- **קטעי משוב:** `node build-feedback.js`.
- **הקלטה אישית** (למשל בקול ההורה): שומרים קובץ שמע ב-`recordings/` לפי השמות
  שמוסברים ב-`recordings/README-שמות-קבצים.md` (למשל `ב.m4a` לשם האות, `כל הכבוד.m4a`
  למשוב), ואז מריצים `node merge-recordings.js`. ההקלטה מוטמעת מעל קטע ה-TTS.
- אחרי כל שינוי בקול, מגדילים את מספר הגרסה ב-`index.html` בשורה
  `<script src="audio-data.js?v=N">` כדי לעקוף מטמון, ואז `git push`.

### הערות
- זיהוי דיבור/מיקרופון עובד רק ב-HTTPS (כמו GitHub Pages) או `localhost`, לא בפתיחת
  קובץ מקומי (`file://`). לכן האתר החי הוא הדרך המומלצת.
- הכלים דורשים Node.js (גרסה 18+).
