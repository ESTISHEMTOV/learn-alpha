# 🌈 הא-ב הקסום

אפליקציה צבעונית ללימוד ה-א-ב לילדים בגיל הרך — עם הקראה קולית בעברית, ציורים,
משחקים, ומערכת ניקוד וגביעים. עובדת בכל דפדפן, גם ללא התקנה.

## מה יש באפליקציה
- **🔤 לומדים אותיות** — כרטיסייה לכל אות עם ציור, מילה, והקראה בעברית.
- **🎨 כל האותיות** — לוח כל הא-ב; לחיצה מקריאה את האות והמילה.
- **👂 מצא את האות** — שומעים אות ובוחרים אותה.
- **🖼️ איזו אות ראשונה?** — רואים ציור ובוחרים באיזו אות הוא מתחיל.
- **🗣️ תגידי את האות** — אומרים את האות בקול ובודקים (עם אפשרות מיקרופון).
- **🏆 האליפות שלי** — צוברים נקודה על כל הצלחה; ב-100 נקודות מקבלים גביע.

## הרצה
פשוט פותחים את `index.html` בדפדפן. באתר החי (GitHub Pages) הכל עובד אוטומטית,
כולל זיהוי הדיבור והרשאת מיקרופון שנזכרת.

## קבצים
- `index.html` — האפליקציה כולה (HTML/CSS/JS).
- `audio-data.js` — קטעי קול עבריים מוטמעים (base64).
- `recordings/` — הקלטות קול אישיות שמחליפות קטעים (ראו הסבר שם).
- `build-audio.js` / `merge-recordings.js` — כלים לבניית/מיזוג קטעי הקול.
- `server.js` / `start-server.bat` — שרת מקומי אופציונלי להרצה מ-localhost.
