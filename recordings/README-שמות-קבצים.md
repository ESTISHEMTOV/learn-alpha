# הקלטות קול — איך לתת שם לקבצים

שמרי כל הקלטה בתיקייה הזו (`recordings`). פורמט: m4a / mp3 / wav / ogg — הכל עובד.

## הכי פשוט (מומלץ)
- **שם האות** → קראי לקובץ בשם האות עצמה. דוגמה: `ב.m4a` = שם האות "בֵּית".
- **המילה** → שם האות + רווח + "מילה". דוגמה: `ב מילה.m4a` = המילה "בננה".

## אפשרות נוספת (לפי מספר)
`L02` = שם האות ב, `W02` = המילה של ב (המספר = מיקום האות בא-ב).

אפשר להקליט רק מה שרוצים — כל השאר יישאר עם הקול הקיים.
כשמסיימים, פשוט כותבים לי "סיימתי" ואני מטמיע ומעדכן.
